Hello, I'm a programmer and I've been tasked with creating a simple chatbot that can understand and respond to basic queries. I've been using the NLTK library for natural language processing tasks. I've implemented a basic chatbot that can understand and respond to some simple queries, but I'm having trouble with more complex queries. The bot is currently able to understand the basic queries, but it doesn't seem to be able to handle the more complex queries.

One of the issues I'm experiencing is that the bot's response is not being generated dynamically based on the user's input, it's always returning the same response for every query.

Here is a simple example of the code:

```python
import nltk
from nltk.stem import WordNetLemmatizer
import random

# Initialize NLTK data
nltk.download('wordnet')
lemmatizer = WordNetLemmatizer()

class Chatbot:
    def __init__(self):
        self.responses = {
            "hello": ["Hi, how are you?", "Hello! What's up?", "Hi, I'm here to help you!"]
        }

    def process_input(self, input_str):
        # Tokenize the input string
        tokens = nltk.word_tokenize(input_str)

        # Lemmatize the tokens
        lemmatized_tokens = [lemmatizer.lemmatize(token) for token in tokens]

        # Check if the lemmatized tokens are in the chatbot's responses
        for key, value in self.responses.items():
            if lemmatized_tokens in value:
                return value

        # If the lemmatized tokens are not in the chatbot's responses, return a default response
        return "I didn't understand that."

# Test the chatbot
chatbot = Chatbot()
print(chatbot.process_input("hello"))  # Should return "Hi, how are you?"
print(chatbot.process_input("what's up"))  # Should return "Hi, I'm here to help you!"
print(chatbot.process_input("I'm feeling sad"))  # Should return "I didn't understand that."
```
