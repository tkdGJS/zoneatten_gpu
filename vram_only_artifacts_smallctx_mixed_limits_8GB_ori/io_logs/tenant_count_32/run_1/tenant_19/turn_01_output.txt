In the following text, which statement is correct about the relation between the number of atoms in a molecule of a substance, the number of electrons in a molecule, and the number of protons in a nucleus.

## Step 1: Understand the basic principles of atomic structure
Atoms are the building blocks of matter, and they consist of protons, neutrons, and electrons. The number of protons in an atom's nucleus determines the element of an atom, while the number of electrons equals the number of electrons in a neutral atom. The number of neutrons can vary, but it is always less than the number of protons.

## Step 2: Relate the number of atoms to the number of electrons in a molecule
A molecule is formed when atoms bond together. The total number of electrons in a molecule is equal to the sum of the number of electrons in each individual atom that makes up the molecule. Since each atom contributes its electrons to form the molecule, the total number of electrons in a molecule is equal to the number of atoms in the molecule.

## Step 3: Relate the number of atoms to the number of protons in a nucleus
The number of protons in a nucleus is a fixed number for each element, known as the atomic number, and is a characteristic of the element itself. It does not change regardless of the number of atoms in a molecule.

## Step 4: Consider the relationship between the number of atoms and the number of electrons in a molecule
Since the number of atoms in a molecule equals the number of electrons in a molecule, and the number of protons in a nucleus is fixed for an element, the statement that the number of atoms in a molecule is equal to the number of electrons in a molecule is correct.

The final answer is: $\boxed{yes}$
